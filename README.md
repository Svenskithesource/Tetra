A basic interpreted language, written in Python!
### Installation
You can clone the repo and then install it locally.
```bash
git clone https://github.com/Svenskithesource/Tetra
cd Tetra
pip3 install -e .
```

### Usage
`tetra -h`

### Tests
To run the tests do:

`python -m unittest tests.test_tetra`
